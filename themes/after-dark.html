after-dark

Secure and accessible dark theme

Live demo

To use this theme, simply clone it into your themes directory:

cd themes && git clone https://github.com/grego/after-dark

Or use it as a submodule:

cd themes && git submodule add https://github.com/grego/after-dark

Then, set it in your Blades.toml:

theme = after-dark
